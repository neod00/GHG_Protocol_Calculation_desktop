# 다운로드 페이지 문구 템플릿 (무서명 베타)

## 상단 배너

`GHG Desktop 베타 버전 안내`  
이 버전은 초기 베타이며 Windows 코드 서명이 아직 적용되지 않았습니다. 설치 시 SmartScreen 경고가 표시될 수 있습니다.

## 제품 소개 문구

GHG Desktop은 Scope 1, 2 배출량 계산과 보고서 생성을 위한 로컬 우선 데스크탑 앱입니다.  
입력 데이터는 기본적으로 사용자 PC에 저장됩니다.

## 설치 파일 섹션

- 버전: `v0.1.0-beta.1`
- 운영체제: `Windows x64`
- 설치 파일: `GHG-Desktop-0.1.0-beta.1-x64.msi`
- 해시(SHA-256): `<여기에_해시값>`
- 배포일: `<YYYY-MM-DD>`

## 설치 방법

1. 설치 파일을 다운로드합니다.
2. 파일을 실행합니다.
3. Windows 보호 화면이 표시되면 `추가 정보`를 클릭합니다.
4. `실행`을 클릭해 설치를 진행합니다.
5. 설치 후 앱을 실행하고 라이선스 키를 입력합니다.

## 업데이트 안내

- 앱은 시작 시 최신 버전을 확인합니다.
- 중요 보안/정책 업데이트는 강제 업데이트로 적용될 수 있습니다.
- 강제 업데이트 대상 버전에서는 계속 사용이 제한될 수 있습니다.

## 라이선스 안내

- 라이선스 상태에 따라 일부 기능이 제한될 수 있습니다.
- 결제 미납 또는 정책 위반 시 사용이 차단될 수 있습니다.
- 문의: `<지원 이메일>`
